Basic UI Kit

Um UI Kit multi-marca escalável com acessibilidade em seu núcleo.

Ver projeto no Figma

Sistema multi-marca

Esse não é mais um UI Kit. É um sistema multi-marcas flexível criado a partir de uma única fonte de verdade. Cada marca mantém sua identidade única enquanto compartilha a mesma arquitetura de componentes.

Design tokens

Sistema baseado em tokens garantindo consistência e escalabilidade. Cada decisão de design é armazenada como um token reutilizável, tornando as atualizações perfeitas em todo o sistema.

Cores

Cores primárias, secundárias e semânticas com variantes claro e escuro. Todas as combinações de cores são testadas para conformidade com acessibilidade.

Acessibilidade & modo escuro

Construído com as diretrizes WCAG em mente. Cada combinação de cores é testada para proporções de contraste, e ambos os modos claro e escuro são projetados para serem igualmente acessíveis.

WCAG 2.1 AA Cores acessíveis Pronto para modo escuro

Sistema de grid responsivo

Um sistema de grid flexível projetado para aplicações do mundo real. Os componentes se adaptam perfeitamente do desktop ao mobile, mantendo harmonia visual em todos os breakpoints.

componentes projetados para escalabilidade

A atenção aos detalhes diferencia este UI Kit. Cada componente inclui estados e variações abrangentes para lidar com cenários do mundo real.

Fluxos de usuário

Veja como o UI Kit se traduz em experiências reais de produto. Estes fluxos demonstram aplicações práticas da biblioteca de componentes.

Produtos

A tela de produtos exibe todos os itens cadastrados com informações importantes como preço, categoria e desempenho de vendas.

Novo produto

Adicionar um novo produto é simples com um formulário limpo que inclui todos os campos necessários: nome, modelo, fabricante, categoria, preço, fotos e especificações.

Produto adicionado

Após adicionar um produto com sucesso, os usuários recebem uma notificação de confirmação com um link rápido para visualizar o item recém-criado.

Informações do produto

A visualização detalhada do produto exibe todas as informações, incluindo múltiplas fotos, especificações e uma opção de edição para atualizações rápidas.
